Users Table

Method Path Description
GET / Get all users
GET /users/:id Gets single user by userID
POST /register Adds a one user
PATCH/PUT /update/:id Edits user details by userID
DELETE /delete/:id Delete user by userID
POST /login Signs a user into their account

Products Table

Method Path Description

GET /:id Gets single item by itemID
POST /items/add Adds a single item
PATCH /update/:id Edits and updates a single items details by itemID
DELETE /delete/:id Delete item by itemID
